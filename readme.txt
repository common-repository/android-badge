Mit diesem Plugin kannst du Informationen und Screenshots von Android Apps in Seiten und Artikeln anzeigen lassen.

== Description ==

*Dieses Plugin ermöglicht dir die Darstellung von Informationen einer Android App von Google Play.*

Nachdem du dieses Plugin installiert hast, kannst du den folgenden Code in einen Artikel oder eine Seite einfügen:

[appbadge id="com.google.android.apps.maps"]

Ersetze dabei die id com.google.android.apps.maps durch die id der gewünschten App (in der URL bei Google Play enthalten).

<span style=\"text-decoration: underline;\">Die wichtigsten Funktionen im Überblick:</span>

*   Badge enthält allgemeine Informationen wie Entwickler, -url, Kategorie und die Beschreibung
*   Screenshots und Icons werden angezeigt
*   Informationen zum Download
*   Breite wird automatisch angepasst

<span style=\"text-decoration: underline;\">Beispiel:</span>
Weitere Informationen und ein Beispiel finden sich unter http://de.merq.org/android-appbadge-example/.

== Installation ==

1. Installiere das Plugin im Admin-Bereich oder extrahiere die ZIP-Datei, lade sie in den Ordner wp-content/plugins/ hoch und aktiviere das Plugin im Admin-Bereich.
2. Nun kannst du Artikel mit dem Shortcode [appbadge id="ID"] erstellen.
== Frequently Asked Questions ==

= Woher bekomme ich die notwendige id einer App? =

1. Rufe bei Google Play eine App auf.
2. Sehe dir die URL an, z.B. https://play.google.com/store/apps/details?id=com.google.android.apps.maps&feature=...
3. Kopiere die id; in diesem Beispiel: com.google.android.apps.maps
